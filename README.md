Tá na Mão — Aplicativo de Serviços
Aplicativo Android desenvolvido em Kotlin com o objetivo de conectar clientes e trabalhadores, permitindo o gerenciamento e solicitação de serviços de forma simples e organizada.
O projeto foi desenvolvido como parte de estudos em Desenvolvimento Mobile, focando na criação de uma aplicação funcional com autenticação de usuários, armazenamento local e interface interativa.

Objetivo do Projeto:
Este projeto foi criado com fins educacionais e de portfólio, com o objetivo de praticar:
- Desenvolvimento Android
- Estruturação de aplicações mobile
- Persistência de dados com Room
- Implementação de interfaces interativas

Funcionalidades:
- Sistema de login e cadastro
- Diferenciação entre Usuário e Trabalhador
- Validação de login
- Agendamento de serviços
- Listagem de serviços disponíveis
- Gerenciamento de solicitações
- Interface otimizada para dispositivos Android

Tecnologias Utilizadas:
- Kotlin
- Android Studio
- Room Database

Principais conceitos utilizados:
- DAO (Data Access Object) para manipulação do banco de dados
- Entities para representação das tabelas
- RecyclerView para listagem dinâmica de dados

Instalação e Execução:
- Clone o repositório: git clone https://github.com/Marley-Z/App-Ta-Na-Mao
- Abra o projeto no Android Studio
- Aguarde o Gradle sincronizar
- Execute o aplicativo em emulador ou dispositivo físico
